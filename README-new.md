# medusa-fulfillment-shippo

Adds [Shippo](https://goshippo.com/) as a fulfillment provider in [Medusa](https://medusajs.com/)

Medusa is an open-source headless commerce engine that enables developers to create amazing digital commerce experiences.
  
## Getting Started
  
cc

A Medusa server running locally. For quick setup use [medusa-starter-default](https://github.com/medusajs/medusa-starter-default)




### Installing

```
npm install medusa-fulfillment-shippo
```  

Set the options in `medusa-config.js`:
```
{
	resolve:  `medusa-fulfillment-shippo`,
	options: {
		api_key:  SHIPPO_API_KEY,
		weight_unit_type:  'g',
		dimension_unit_type:  'cm'
	},
}
```
